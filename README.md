# Startup Funding Analysis

## Project Overview

This project analyzes multi-year startup funding data to uncover investment trends, identify top-funded cities and industries, and evaluate dominant funding types.

The objective was to transform raw funding records into structured business insights using spreadsheet-based aggregation and interactive dashboard visualization.

---

## Objectives

- Analyze startup funding trends over time  
- Identify top-funded cities  
- Determine leading industry verticals  
- Evaluate dominant investment types  
- Build an interactive dashboard for comparative analysis  

---

## Dataset

The dataset includes startup funding records across multiple years, containing:

- Startup name  
- City location  
- Industry vertical  
- Investment type  
- Funding amount (USD)  
- Funding date  

---

## Tools & Technologies

- **Google Sheets** – Data cleaning, preprocessing, pivot table analysis  
- **Tableau Public** – Interactive dashboard creation and visualization  

---

## Data Cleaning & Preparation

The following preprocessing steps were performed:

- Removed null and inconsistent records  
- Standardized city names (e.g., Bengaluru → Bangalore)  
- Converted funding amounts into numeric format  
- Aggregated data using pivot tables  

Created pivot tables for:

- Year-wise total funding  
- Top cities by funding  
- Top industries by funding  
- Top investment types by funding  

Structured preprocessing ensured reliable aggregation and analysis.

---

## Dashboard & Visualizations

An interactive Tableau dashboard was built containing:

### Funding Trend (Line Chart)
- Displays total startup funding across years  
- Highlights investment growth patterns and funding cycles  

### Top Cities (Bar Chart)
- Identifies major startup funding hubs  
- Compares funding concentration geographically  

### Top Industries (Bar Chart)
- Shows which industry verticals attracted the highest investments  

### Top Investment Types (Bar Chart)
- Analyzes dominant funding rounds (e.g., Private Equity, Series B)

All visualizations were integrated into a single interactive dashboard for streamlined comparative analysis.

---

## Key Insights

- Startup funding peaked in specific years, indicating cyclical investment behavior.  
- Bangalore emerged as the leading startup funding hub.  
- E-Commerce and Consumer Internet attracted the highest investments.  
- Private Equity and Series B rounds dominated total funding volume.
